# AelitaOS Alpha 0.1 Target

Alpha 0.1 is the first cautious smoke-test build path for Xiaomi 11 Lite 5G NE (`lisa`). It is not final AelitaOS and is not ready to flash from this repository.

## Target outcome

- First bootable Android-based test ROM path for `lisa`.
- Aelita-Fon available as a preinstalled/preloaded app or system app candidate.
- Aelita can be selected as the HOME launcher.
- Shell opens.
- System Agent opens.
- Local Core works.
- App list/search/launch works through normal Android APIs.
- Permission Center works.

## Explicit non-goals for Alpha 0.1

- No privileged permissions.
- No Accessibility.
- No Notification Listener.
- No Usage Stats.
- No background service.
- No framework hooks.
- No root.
- No external AI API.
- No cloud backend.

Alpha 0.1 is a smoke-test build path only. It proves the base ROM path and Aelita-Fon inclusion path before deeper system integration is attempted.
