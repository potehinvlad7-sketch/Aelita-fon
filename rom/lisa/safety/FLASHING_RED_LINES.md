# Flashing Red Lines

- Never flash files for another codename.
- Never flash if codename is not `lisa`.
- Never flash unknown ZIPs.
- Never flash without a stock restore path.
- Never lock bootloader after custom ROM.
- Never erase partitions blindly.
- Never run random scripts.
- Never mix firmware/vendor regions without research.
- Never claim success without first boot checklist.
- Never use daily phone without backup.
